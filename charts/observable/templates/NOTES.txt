ROLLBACK

List revisions:
  helm history {{ .Release.Name }} --namespace {{ .Release.Namespace }}

Roll back to a previous revision:
  helm rollback {{ .Release.Name }} <revision> --namespace {{ .Release.Namespace }}

Note: schema migrations are forward-only (ADR-013).  Rolling back the
application tier is safe as long as the previous image version is compatible
with the current schema.  Do not re-run the migration Job after rollback.
