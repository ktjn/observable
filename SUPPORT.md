# Support

## Getting help

- **Bug reports and feature requests:** Open an issue using the appropriate
  [issue template](https://github.com/ktjn/observable/issues/new/choose).
- **Security vulnerabilities:** Follow the process in [SECURITY.md](SECURITY.md) — do not open
  a public issue.
- **Questions and discussions:** Use
  [GitHub Discussions](https://github.com/ktjn/observable/discussions) for general questions,
  deployment help, and community conversation.

## Before opening an issue

1. Search [existing issues](https://github.com/ktjn/observable/issues) to avoid duplicates.
2. Include the Observable version (or commit SHA), deployment method (Compose or Helm), and
   relevant logs or reproduction steps.

## Scope

Observable is at an early (0.x) release stage. Maintainers prioritize security fixes and release
blockers tracked in [ROADMAP.md](ROADMAP.md). Feature requests are welcome but may be deferred
until the current milestone is complete.
